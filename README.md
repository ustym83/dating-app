# 🗺️ Heartbeat - Dating Application

Heartbeat is a modern, responsive conceptual web application for dating and social networking. Visually inspired by popular platforms like Tinder, it features a clean user interface, swipe/match mechanics, and integrated modern UI components like a dynamic Light/Dark mode toggle.

## 🚀 Features

* **Discover Dashboard:** Clean card-based user profiles featuring photos, biography, location, and tags.
* **Match Mechanics:** Intuitive visual buttons for liking (Heart) or passing (X) on profiles.
* **Theme Switcher:** Custom-designed, smooth animated Light/Dark mode switch integrated into the navigation.
* **Responsive Layout:** Optimized for mobile-first views, mimicking a native smartphone application experience.
* **Chat System:** Dedicated navigation tab for user conversations and messaging.

## 📸 Interface Preview

The application interface features a clean, card-based discovery view with modern typography and a fluid theme switcher:

* **Light Mode:** High contrast layout optimal for daytime browsing.
* **Dark Mode:** Sleek, low-light interface featuring a dark glassmorphism track with a deep blue toggle thumb.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Flexbox/Grid, CSS Transitions), JavaScript (ES6+)
* **Styling/Architecture:** Mobile-first responsive design, BEM methodology for clean CSS classes.
* **Icons:** SVG vector graphics for pixel-perfect rendering across all screen resolutions.
